# Useage

```
yarn
yarn start
```
or with `npm`
```
npm install
npm start
```

Runs the app in the development mode.
Open http://localhost:8080 to view it in the browser.


## Testing

```
yarn test
```
or
```
npm test
```

Launches the application test runner.
Run with the `--watch` flag (`npm test -- --watch`) to run in interactive watch mode.
